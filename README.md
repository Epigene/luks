# Luks

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'luks' # and bundle
```

## Usage

```rb
# Anywhere in code
red "This thing failed!"
green "Hooray, something was successful"
```

Discover all implemented color methods by running

```rb
Luks.colors
```
